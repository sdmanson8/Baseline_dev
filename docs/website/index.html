Baseline

A modular PowerShell toolkit for configuring, auditing, hardening, and automating Windows 10 & 11 environments.

View on GitHub

🚀 Overview

Baseline is a manifest-driven Windows configuration toolkit featuring a full WPF GUI, compliance tracking, audit trails, preset-based execution, and enterprise deployment support.

🔥 Core Capabilities

GUI + Preview Engine

Visual diff, risk labels, filters, and pre-flight validation before execution.

Compliance Tracking

Drift detection, system snapshots, and audit trails for configuration control.

Preset System

Minimal → Advanced tiers with structured risk and behavior guarantees.

Automation

Headless execution, scheduled tasks, and multi-machine targeting.

⚙️ Presets

  • Minimal – Safe Mode starting point
  • Basic – Recommended default
  • Balanced – Broader tuning
  • Advanced – Expert-only hardening

▶️ Quick Start

.\run.cmd

.\Baseline.ps1 -Preset Basic

🧩 Project Structure

  • Module/ – Core engine, GUI, and tweak modules
  • Module/Data – Manifest-driven configuration
  • Module/GUI – WPF UI architecture
  • Module/Regions – Feature/tweak implementations
  • Tools/ – Validation and build tooling
  • Tests/ – Unit, integration, and fixtures
